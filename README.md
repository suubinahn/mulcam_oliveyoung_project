## 👩‍💻 담당 역할 (My Contribution)

본 프로젝트는 3인 팀 프로젝트로 진행되었으며, 저는 **아이디어 기획과 텍스트 마이닝 및 감성 분석 파트**를 담당했습니다.

---
## mulcam_oliveyoungproject
## [주간프로젝트] 포털사이트 사용자 의견(리뷰)을 활용한 감성 분석
## 프로젝트 소개
올리브영 리뷰 감성분석을 주제로 하여, 데이터를 기반으로 한 생리용품 가이드북(Streamlit)을 제작해보고자 했다. 
## 프로젝트 진행 순서
- 주제 선택
- 크롤링
- 데이터 전처리
- EDA
- Tokenize & tf-idf
- 인사이트
- Streamlit 제작
